# Clay Classification using Machine Learning

## Objective
Classify different clay types based on experimental properties.

## Pipeline
1. Data generation (experimental dataset)
2. Preprocessing (scaling, encoding)
3. Modeling (Random Forest, SVM)
4. Evaluation (accuracy, confusion matrix)
5. Visualization (feature importance)
6. Robustness check (noise perturbation)

## How to Run
- Open the notebook in /notebooks
- Run all cells
